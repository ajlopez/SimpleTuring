# SimpleTuring

Simple Turing Machine implemented in JavaScript that runs in browser and Node.js.


## Installation

Via npm on Node:

```
npm install simpleturing
```


## Usage

Reference in your program:

```js
var simpleturing = require('simpleturing');
```
